Angular Setup

After Installing Vscode and NodeJs on your systems, it's time for us to install angular.

To install Angular follow the steps below:

  1. Open VsCode and open terminal which is located on the top.

  2. Check Whether the NodeJs is installed Successfully.

    • To check run the command in the terminal "npm -version"

    • It displays the version you installed.

  3. Once installation is done successfully. Run the following command in the terminal to install Angular.

    • cmd : "npm install -g @angular/cli"

  4. After that to confirm successful installation,run the following command

    • cmd : "ng version"
