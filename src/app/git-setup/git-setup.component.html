Git Setup

To setup or download Git on your system follow the steps mention below

  1. Click on the link to go to the NodeJs download page Click Here Or Copy the link and paste in the browser "https://git-scm.com/downloads".

  2. Select the Type of your os and the system type and click on it
  3. The download will automatically start. It downloads a .Zip file. Extract The file and start intsllation.

After Successfully Installing Git on your system, login with your Git credentials.
