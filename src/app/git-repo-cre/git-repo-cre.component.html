Git Repository Creation

Login to GitHub qith your credentials.

  1. Clicl on Repository and click on new Repository.

  2. Follow the given Steps below after creating a Repository.

  3. To initialize , add, push, commit changes the below commands will help:

    1. cmd : "git init" (to start git)

    2. cmd : "git add ." (to add all the files into the repo)

    3. cmd : "git commit -m "message_of_commit" (modify the message_of_commit with proper message_of_commit)

    4. cmd : "git push" (to push the code into git)
