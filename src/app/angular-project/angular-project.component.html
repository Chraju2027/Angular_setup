Angular Project Creation

To create an Angular project on your system follow the steps mention below

  1. Click on the file menu in Vscode and there you can either choose to create a new project or you can add an existing project to the workplace.

  2. To create a new project open Vscode terminal and type the command

    • cmd : "ng new project_name (replace the project_name with the name of your project)"

  3. Once the project is created, we can check or run the project by the command

    • cmd : "ng server" (in the terminal)

  4. open the app in localhost:4200
